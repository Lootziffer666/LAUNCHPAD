# LAUNCHPAD Design Principles

These principles govern the controller-first TV surface. Every UI decision must pass through them.

---

## 1. Controller-First

The primary input device is a gamepad, not a mouse. Every interactive element must be reachable, operable, and visually clear via directional navigation and a single confirm button. Keyboard navigation (arrow keys + Enter) is the test proxy.

## 2. Couch Distance Readable

The screen is viewed from 2-3 meters away on a living room TV. Body text must be at least 18px, titles at least 32px. Nothing small, nothing subtle. If you squint, it failed.

## 3. Big Targets

Interactive elements have a minimum touch/focus area of 80x80px. No tiny icons, no inline text links, no desktop-sized buttons. A child with a gamepad must never miss.

## 4. Warm-White / Calm Library Surface

The visual tone is a warm reading room, not a cold tech dashboard. Colors lean toward cream, amber, sage, and terracotta. No pure white, no pure black, no cold blue defaults. Think: sunlit wooden shelf, not server rack.

## 5. Game Art May Carry Emotion

Cover art, screenshots, and game imagery are allowed to be colorful and expressive. The UI frame stays neutral and warm so the art can breathe without competing with interface chrome.

## 6. UI Itself Stays Simple

Minimal chrome. Large type. Generous whitespace. One clear action per screen state. The interface disappears so the content can speak.

## 7. No Dashboard

There is no analytics panel, no statistics overview, no "engagement metrics" view. Parents see a calm library with clear status indicators, not a control center.

## 8. No Dense Panels

No multi-column settings pages, no cramped lists, no scrolling option grids. Each view shows one concern at a time with plenty of breathing room.

## 9. No Store Noise

No advertisements, no "recommended for you", no purchase prompts, no social pressure. This is a curated family library, not a marketplace.

## 10. No Tiny Desktop Controls

No 12px labels, no 32px buttons, no tooltip-dependent icons, no right-click menus, no drag handles. Everything must work at couch scale with a four-button controller.
